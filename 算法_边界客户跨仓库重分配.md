# 算法：边界客户跨仓库重分配

```
输入: 所有仓库集合D, 各仓库路径集合{R_d | d ∈ D}, 距离矩阵Dist, 容量约束Q, 里程约束L, 边界阈值θ
输出: 优化后的路径集合{R_d' | d ∈ D}, 总成本改善Δ_total

1:  Δ_total ← 0, improved ← true
2:  while improved do
3:      improved ← false
4:      for each 仓库对 (d_i, d_j) ∈ D × D, i ≠ j do
5:          BC ← identify_boundary_customers(d_i, d_j, R_{d_i}, Dist, θ)  // 识别边界客户
6:          for each 边界客户 c ∈ BC do
7:              cost_remove ← calculate_removal_cost(c, R_{d_i}, d_i, Dist)  // 移除成本
8:              R_{d_j}', flag, Δ_insert ← 算法2-5(c, d_j, R_{d_j}, Dist, Q, L)  // 使用最佳位置插入
9:              if flag = true and Δ_insert < cost_remove then  // 插入成功且有益
10:                 remove_customer(c, R_{d_i})  // 从原仓库移除
11:                 R_{d_j} ← R_{d_j}'  // 更新目标仓库路径
12:                 Δ_total ← Δ_total + (cost_remove - Δ_insert)
13:                 improved ← true
14:             end if
15:         end for
16:     end for
17: end while
18: return {R_d' | d ∈ D}, Δ_total
```

**说明**: 该算法识别边界客户并尝试跨仓库重分配。第8行调用算法2-5执行最佳位置插入，自动处理容量和里程约束。时间复杂度O(|D|² × |BC| × |R_d| × n²)。

## 边界客户识别函数

```
函数 identify_boundary_customers(d_i, d_j, R_{d_i}, Dist, θ):
输入: 源仓库d_i, 目标仓库d_j, 源仓库路径集合R_{d_i}, 距离矩阵Dist, 边界阈值θ
输出: 边界客户集合BC

1:  BC ← ∅
2:  for each 路径 r ∈ R_{d_i} do
3:      for each 客户 c ∈ r do
4:          if Dist[d_j][c] < Dist[d_i][c] × (1 + θ) then  // 到目标仓库距离接近
5:              BC ← BC ∪ {c}
6:          end if
7:      end for
8:  end for
9:  return BC
\begin{algorithm}
\caption{最佳位置插入算法}
\begin{algorithmic}[1]
\Require 待插入客户$c$, 目标仓库$d$, 路径集合$R_d$, 距离矩阵$Dist$, 容量约束$Q$, 里程约束$L$
\Ensure 更新后路径集合$R_d'$, 插入标志$flag$, 增量成本$\Delta_{best}$

\State $\Delta_{best} \gets +\infty$, $best\_route \gets null$, $best\_pos \gets null$
\For{each $r \in R_d$}
    \If{$\sum_{i \in r} demand[i] + demand[c] > Q$}
        \State \textbf{continue}  \Comment{容量约束检查}
    \EndIf
    \State $dist_r \gets$ calculate\_distance$(r, d, Dist)$
    \For{$pos \gets 1$ to $|r| + 1$}
        \State $r' \gets$ insert$(r, c, pos)$
        \State $dist_{r'} \gets$ calculate\_distance$(r', d, Dist)$
        \If{$dist_{r'} > L$}
            \State \textbf{continue}  \Comment{里程约束检查}
        \EndIf
        \State $\Delta \gets dist_{r'} - dist_r$
        \If{$\Delta < \Delta_{best}$}
            \State $\Delta_{best} \gets \Delta$, $best\_route \gets r$, $best\_pos \gets pos$
        \EndIf
    \EndFor
\EndFor

\If{$best\_route \neq null$}
    \State insert$(best\_route, c, best\_pos)$, $flag \gets true$  \Comment{在最佳位置插入}
\ElsIf{$demand[c] \leq Q$ \textbf{and} $2 \times Dist[d][c] \leq L$}
    \State $R_d \gets R_d \cup \{[c]\}$, $\Delta_{best} \gets 2 \times Dist[d][c]$, $flag \gets true$  \Comment{创建新路径}
\Else
    \State $flag \gets false$  \Comment{插入失败}
\EndIf

\State \Return $R_d$, $flag$, $\Delta_{best}$
\end{algorithmic}
\end{algorithm}
```

**说明**: 边界客户定义为到目标仓库的距离接近当前仓库的客户。参数θ控制边界范围（通常取0.1-0.2）。
