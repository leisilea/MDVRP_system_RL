# RouteFinder 环境修复指南

**当前环境问题**:
1. ❌ PyTorch 版本: 2.3.1 (需要 >= 2.5.0)
2. ⚠️ NumPy 版本: 2.2.6 (与某些模块不兼容，建议降级到 < 2.0)

**当前正常的包**:
- ✅ Python: 3.10.19
- ✅ TensorDict: 0.11.0
- ✅ TorchRL: 0.11.1
- ✅ RL4CO: 0.6.0
- ✅ Lightning: 2.6.1
- ✅ CUDA: 12.1 (RTX 3060)

---

## 方案 1: 在当前环境中升级（快速修复）⚡

### 步骤 1: 升级 PyTorch 到 2.5.0+

```powershell
# 卸载旧版本
pip uninstall torch torchvision torchaudio -y

# 安装 PyTorch 2.5.0+ (CUDA 12.1)
pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu121
```

### 步骤 2: 降级 NumPy

```powershell
# 降级到 NumPy 1.x
pip install "numpy<2.0"
```

### 步骤 3: 验证

```powershell
python RL4CO_Integration/check_routefinder_env.py
```

**优点**: 快速，不需要重建环境  
**缺点**: 可能有依赖冲突

---

## 方案 2: 创建全新的 Conda 环境（推荐）⭐

### 完整命令（复制粘贴执行）

```powershell
# 1. 停用当前环境
conda deactivate

# 2. 创建新环境
conda create -n routefinder python=3.12 -y

# 3. 激活新环境
conda activate routefinder

# 4. 安装 PyTorch 2.5.0+ (CUDA 12.1)
conda install pytorch torchvision torchaudio pytorch-cuda=12.1 -c pytorch -c nvidia -y

# 5. 安装 NumPy 1.x
pip install "numpy<2.0"

# 6. 克隆 RouteFinder（如果还没有）
git clone https://github.com/ai4co/routefinder
cd routefinder

# 7. 安装 RouteFinder（会自动安装 RL4CO 等依赖）
pip install -e .

# 8. 验证安装
cd ..
python RL4CO_Integration/check_routefinder_env.py
```

**优点**: 干净的环境，不会有冲突  
**缺点**: 需要重新下载安装

## 方案 3: 使用 environment.yml 一键创建

### 步骤 1: 创建修正后的 environment.yml

保存以下内容为 `routefinder_fixed.yml`:

```yaml
name: routefinder
channels:
  - pytorch
  - nvidia
  - conda-forge
  - defaults
dependencies:
  - python=3.12
  - pytorch>=2.5.0
  - torchvision
  - torchaudio
  - pytorch-cuda=12.1
  - pip
  - git
  - pip:
    - "numpy<2.0"
    - tensordict>=0.6.0
    - torchrl>=0.6.0
    - lightning>=2.1.0
    - huggingface-hub[cli]>=0.31.2
    - einops
    - hydra-core
    - hydra-colorlog
    - omegaconf
    - pyrootutils
    - rich
    - robust-downloader
    - scipy
    - wandb
    - pytest
    - pytest-xdist
    - git+https://github.com/ai4co/rl4co.git@main
```

### 步骤 2: 创建环境

```powershell
# 停用当前环境
conda deactivate

# 使用 yml 创建环境
conda env create -f RL4CO_Integration/routefinder_fixed.yml

# 激活环境
conda activate routefinder

# 克隆并安装 RouteFinder
git clone https://github.com/ai4co/routefinder
cd routefinder
pip install -e .
```

## 详细执行步骤（方案 2）

### 第 1 步: 准备工作

```powershell
# 查看当前环境
conda env list

# 停用当前环境
conda deactivate
```

### 第 2 步: 创建新环境

```powershell
# 创建 Python 3.12 环境
conda create -n routefinder python=3.12 -y

# 激活环境
conda activate routefinder

# 确认 Python 版本
python --version
# 应该显示: Python 3.12.x
```

### 第 3 步: 安装 PyTorch

```powershell
# 安装 PyTorch 2.5.0+ (CUDA 12.1)
conda install pytorch torchvision torchaudio pytorch-cuda=12.1 -c pytorch -c nvidia -y

# 验证 PyTorch
python -c "import torch; print(f'PyTorch: {torch.__version__}'); print(f'CUDA: {torch.cuda.is_available()}')"
# 应该显示: PyTorch: 2.5.x, CUDA: True
```

### 第 4 步: 安装 NumPy 1.x

```powershell
# 安装 NumPy < 2.0
pip install "numpy<2.0"

# 验证 NumPy
python -c "import numpy; print(f'NumPy: {numpy.__version__}')"
# 应该显示: NumPy: 1.x.x
```

### 第 5 步: 克隆 RouteFinder

```powershell
# 如果已经克隆过，跳过这步
# 如果没有，执行：
git clone https://github.com/ai4co/routefinder
cd routefinder
```

### 第 6 步: 安装 RouteFinder

```powershell
# 在 routefinder 目录下执行
pip install -e .

# 这会自动安装：
# - RL4CO (main 分支)
# - TensorDict
# - TorchRL
# - Lightning
# - 其他所有依赖
```

### 第 7 步: 验证安装

```powershell
# 返回上级目录
cd ..

# 运行检查脚本
python RL4CO_Integration/check_routefinder_env.py

# 应该看到所有 ✅
```

### 第 8 步: 下载预训练模型（可选）

```powershell
cd routefinder
python scripts/download_hf.py
```

## 常见问题

### Q: 安装 PyTorch 时下载很慢？

使用清华镜像：
```powershell
pip install torch torchvision torchaudio --index-url https://pypi.tuna.tsinghua.edu.cn/simple
```

### Q: conda install 很慢？

添加国内镜像：
```powershell
conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/main
conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/free
conda config --set show_channel_urls yes
```

### Q: 如何删除旧环境？

```powershell
conda deactivate
conda env remove -n GD  # 或者你的旧环境名
```
